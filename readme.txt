Sistema simple de chat funnels para capturar leads y dirigir a WhatsApp.

== Description ==

WAC Chat Funnels es un plugin de WordPress que crea un widget de chat interactivo para capturar leads y dirigir a los usuarios a WhatsApp.

**Características principales:**

* Widget de chat animado con burbuja teaser
* Captura de leads con formulario integrado
* Integración directa con WhatsApp
* Configuración visual simple (sin código)
* Diseño responsive para móvil y desktop
* Sin dependencias externas

**Flujo del chat:**

1. Burbuja teaser aparece después de un delay configurable
2. Mensaje de bienvenida del asistente virtual
3. Opciones de acción (Cotización, WhatsApp, Ver portafolio)
4. Formulario de captura (nombre, email, teléfono)
5. Redirección a WhatsApp con mensaje personalizado

**Fácil de usar:**

1. Instala y activa el plugin
2. Ve a Chat Funnels → Agregar Nuevo
3. Configura el teaser y número de WhatsApp
4. Guarda y el chat aparecerá automáticamente en tu sitio

== Installation ==

1. Sube el plugin a `/wp-content/plugins/wac-chat-funnels/`
2. Activa el plugin desde el menú 'Plugins' en WordPress
3. Ve a 'Chat Funnels' en el menú de administración
4. Crea tu primer funnel
5. Configura y guarda
6. ¡El chat aparecerá en tu sitio web!

== Frequently Asked Questions ==

= ¿Necesito configurar algo especial? =

No, el plugin funciona inmediatamente después de la activación. Solo necesitas crear un funnel y configurar tu número de WhatsApp.

= ¿El chat funciona en móviles? =

Sí, el widget es completamente responsive y se adapta a todos los dispositivos.

= ¿Puedo personalizar los mensajes? =

Por ahora los mensajes están predefinidos para mantener la simplicidad. En futuras versiones se podrán personalizar.

= ¿Los datos de los leads se guardan? =

Sí, los datos se capturan y se envían automáticamente a WhatsApp. También se pueden almacenar en la base de datos.
