Sistema de chat funnels conversacionales para capturar leads y dirigir a WhatsApp

== Description ==

WAC Chat Funnels es un plugin de WordPress que te permite crear chatbots conversacionales para capturar leads y dirigir tráfico a WhatsApp. Perfecto para agencias y empresas que quieren automatizar la generación de leads.

= Características principales: =

* **Widget de chat flotante** - Se integra perfectamente en tu sitio web
* **Editor YAML visual** - Crea funnels complejos sin programación
* **Reglas de targeting avanzadas** - Muestra el chat solo cuando sea relevante
* **Integración con WhatsApp** - Dirige leads directamente a WhatsApp
* **Analytics integrado** - Rastrea conversiones y rendimiento
* **Webhooks y email** - Envía leads a tu CRM o email
* **Responsive design** - Funciona en móviles y desktop
* **Multi-idioma** - Soporte para múltiples idiomas

= Casos de uso: =

* **Captura de leads** - Formularios conversacionales
* **Soporte al cliente** - FAQ automatizado
* **Ventas** - Dirigir a WhatsApp para cerrar ventas
* **Marketing** - Segmentación por UTM, página, dispositivo
* **E-commerce** - Asistencia en el proceso de compra

= Integraciones: =

* WhatsApp Business API
* Google Analytics 4
* Meta Pixel
* Webhooks (Zapier, Make, etc.)
* Email automático
* Exportación CSV

== Installation ==

1. Sube el plugin a la carpeta `/wp-content/plugins/wac-chat-funnels/`
2. Activa el plugin desde el panel de WordPress
3. Ve a "Chat Funnels" en el menú lateral
4. Crea tu primer funnel
5. Configura las reglas de targeting
6. ¡Listo! El widget aparecerá en tu sitio

== Privacy Policy ==

Este plugin recopila datos de los usuarios que interactúan con los chat funnels para:
* Procesar leads y contactos
* Analizar el rendimiento de los funnels
* Enviar notificaciones por email

Los datos se almacenan localmente en tu sitio de WordPress y pueden ser exportados o eliminados según la política de privacidad de tu sitio.

== Technical Details ==

* **Frontend**: TypeScript + Web Components + Shadow DOM
* **Backend**: PHP 8.1+ + WordPress REST API
* **Base de datos**: MySQL con tablas personalizadas
* **Storage**: localStorage para estado del usuario
* **Tracking**: GA4, Meta Pixel, analytics interno
* **Bundle size**: < 45KB gzip
* **Compatibilidad**: WordPress 6.0+, navegadores modernos
